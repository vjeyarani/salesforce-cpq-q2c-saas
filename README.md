# Salesforce CPQ – SaaS Quote to Contract Project

## Overview
This project demonstrates an end-to-end Quote-to-Contract implementation
using Salesforce Sales Cloud and Salesforce CPQ for a B2B SaaS business.

## Business Scenario
A SaaS company sells subscription-based products with optional add-ons,
volume-based pricing, and approval-driven discounts.

## Scope
- Sales Cloud foundation
- Salesforce CPQ product configuration
- Pricing & discount rules
- Quote → Contract → Order
- Amendments and Renewals

## Out of Scope
- Billing
- Payments
- Revenue Recognition

## Key Skills Demonstrated
- CPQ Product & Pricing Architecture
- Q2C Process Design
- Salesforce Declarative Automation
- Enterprise-ready Solution Documentation
